## コンテンツ更新フロー

### MSの仕様アップデート対応
1. MS Learnの更新情報やAzure Updateを確認
2. 該当する参考書ページのHTMLを修正
3. 関連する問題の解説を更新（必要に応じて問題自体も修正）
4. Git push → GitHub Pagesに自動反映

### 問題追加フロー
1. Claude Codeで問題を生成（試験範囲・過去問傾向を指示）
2. 生成された問題をレビュー・修正
3. JSONファイルに追加
4. Git push → GitHub Pagesに自動反映
