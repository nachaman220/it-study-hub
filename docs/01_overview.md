# プロジェクト概要

## プロジェクト名
Azure Certification Study Hub

## 目的
Azure Solutions Architect Expert (AZ-305) の学習を支援するブラウザベースの学習サイトを構築する。
従来の「参考書を読む → 過去問を解く」という退屈なフローを改善し、効率的にインプットできる環境を提供する。

## コンセプト
- MS Learnの情報を再構成し、試験対策に特化した参考書ページを提供
- 4択問題と詳細な解説で知識を定着させる
- 将来的に他のクラウド資格・試験にもスケール可能な設計

## スコープ

### 初回リリース（全分野）
- 参考書ページ：全4分野（ID・ガバナンス・監視、データ、事業継続性、インフラストラクチャ）
- 4択問題機能（Claude Codeで生成 → 静的に公開）
- 音声読み上げ機能（Web Speech API）
- 静的HTML/CSS/JSによるブラウザアプリ
- GitHub Pagesでの無料ホスティング

### 実装順序
1. まず「ID・ガバナンス・監視ソリューションの設計」分野を実装
2. ユーザー確認 → 作り方をブラッシュアップ
3. ブラッシュアップを反映して他分野を順次実装

### 将来追加
- 他資格への展開（AWS、GCP、非クラウド資格）
- MSの仕様アップデートへの追従

## 制約
- 追加コストなし（Claude Code Maxプラン以外）
- 進捗管理機能は不要
- AI連携はブラウザ上では行わない（問題生成はオフラインでClaude Codeが実施）
