# UI設計

## デザイン方針
- シンプルで読みやすい。装飾より可読性を優先
- MS Learnのようなテック系ドキュメントの雰囲気
- レスポンシブ対応（PC・スマホ両方で読める）

## レスポンシブ対応
- PC（1024px以上）: サイドバー + メインコンテンツの2カラム
- タブレット/スマホ（1024px未満）: サイドバーを非表示、ハンバーガーメニューに切り替え

## フォント
- 本文: system-ui（OS標準フォント）
- コード・数値: monospace
- 外部フォント読み込みなし（パフォーマンス重視）

## セキュリティUI
- コピー防止: 右クリック・テキスト選択・Ctrl+C/U/S/P 無効化
- プレミアムコンテンツ: ロックアイコン + オーバーレイモーダル表示
- サイドバー: 🔓（無料会員で解放）/ 🔒（有料会員限定）アイコン表示

## 管理者パネル
- Ctrl+Shift+A で表示（ログイン中の管理者のみ）
- 画面右下に固定表示
- 3モード切替: premium / free / guest
- メールアドレスのSHA-256ハッシュで管理者判定
