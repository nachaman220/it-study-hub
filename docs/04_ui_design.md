# UI設計

## デザイン方針
- シンプルで読みやすい。装飾より可読性を優先
- MS Learnのようなテック系ドキュメントの雰囲気
- レスポンシブ対応（PC・スマホ両方で読める）
- ダークモード対応（長時間の学習を想定）

## レスポンシブ対応
- PC（1024px以上）：サイドバー + メインコンテンツの2カラム
- タブレット/スマホ（1024px未満）：サイドバーを非表示、ハンバーガーメニューに切り替え

## フォント
- 本文：system-ui（OS標準フォント）
- コード・数値：monospace
- 外部フォント読み込みなし（パフォーマンス重視）
